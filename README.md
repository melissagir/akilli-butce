
## Canlı Uygulama Linki: https://akilli-butce.streamlit.app/
# Akıllı Bütçe ve Harcama Analizörü

Harcamaları sadece takip etmekle kalmayıp istatistiksel olarak yorumlayan modern bir bütçe uygulaması.

## Özellikler

- 📊 İstatistiksel harcama analizi
- 😊 Mutluluk Skoru entegrasyonu
- 📈 Interaktif grafikler ve trendler
- 🎯 Akıllı "Vibe Check" yorumları
- 🌙 Modern koyu tema arayüz

## Kurulum

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Proje Yapısı

```
akilli_butce/
├── app.py              # Ana Streamlit uygulaması
├── data_manager.py     # Veritabanı yönetimi
├── analyzer.py         # İstatistiksel analiz motoru
├── dashboard.py        # Dashboard bileşenleri
├── requirements.txt    # Python bağımlılıkları
└── README.md          # Proje dokümantasyonu
```
